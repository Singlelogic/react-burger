### Установка проекта
Клонирование репозитория:
 ```bash
 git clone https://github.com/Singlelogic/react-burger.git
 ```
Установка зависимостей:
 ```bash
 npm install
 ```
Запуск проекта
 ```bash
 npm start
 ```

### Тестирование

Используются следующие библиотеки:

* Jest
* Cypress

Запуск тестов:
```bash
npm run test
npm run cypress:open
```
